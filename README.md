## 📖 نبذة

<div dir="rtl">

**قدرة ستوديو** هو أداة احترافية لإنشاء فيديوهات قرآنية مخصصة لمنصتي اليوتيوب والتيك توك. يوفر البرنامج قوالب فيديو احترافية جاهزة، أصوات قراء مشهورين، تصميمات مخصصة لكل منصة، وإمكانية تخصيص كاملة للألوان والخطوط مع معاينة مباشرة قبل التصدير. التطبيق مبني بتقنيات Next.js و TypeScript مع دعم Python لمعالجة الفيديو، مما يضمن نتائج عالية الجودة وسهولة في الاستخدام.

</div>

## 🚀 التشغيل

### المتطلبات

- Node.js 18+
- npm أو yarn أو bun

### التثبيت

```bash
# استنساخ المستودع
git clone https://github.com/ziadamr45/quadra_studio.git
cd quadra_studio

# تثبيت التبعيات
npm install
# أو
bun install

# إعداد متغيرات البيئة
cp .env.example .env
# عدّل ملف .env بإعدادات قاعدة البيانات

# تشغيل تهجيرات قاعدة البيانات
npx prisma migrate dev

# تشغيل خادم التطوير
npm run dev
```

التطبيق سيعمل على `http://localhost:3000`

## English


**Quadra Studio** is a professional tool for creating Quranic videos customized for YouTube and TikTok platforms. The program provides ready-made professional video templates, famous reciter voices, platform-specific layouts, and full customization of colors and fonts with live preview before export. Built with Next.js and TypeScript with Python support for video processing, ensuring high-quality results and ease of use.

### Features

| Feature | Description |
|---------|-------------|
| 🎬 Professional video templates | Diverse templates ready for instant use |
| 🎙️ Famous reciter voices | Audio library of renowned reciters |
| 📐 YouTube & TikTok optimized layouts | Sizes and designs suited for each platform |
| 🎨 Full color & font customization | Complete freedom in customization |
| 📱 Easy-to-use interface | Simple and intuitive design |
| 🔄 Live preview before export | See the result before saving |
| 🌙 Dark/Light mode | Choose your preferred theme |
| 💾 Save projects | Save your work and return to it later |

### Getting Started

#### Prerequisites

- Node.js 18+ or Bun
- npm, yarn, or bun

#### Installation

```bash
# Clone the repository
git clone https://github.com/ziadamr45/quadra_studio.git
cd quadra_studio

# Install dependencies
npm install
# or
bun install

# Set up environment variables
cp .env.example .env
# Edit .env with your database configuration

# Run database migrations
npx prisma migrate dev

# Start development server
npm run dev
```

The app will be available at `http://localhost:3000`
