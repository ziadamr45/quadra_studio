# المساهمة في المشروع | Contributing

شكراً لاهتمامك بالمساهمة! هذا المشروع مفتوح للاقتراحات والتحسينات.
Thank you for your interest in contributing! This project is open to suggestions and improvements.

## كيف تساهم | How to Contribute

1. Fork المشروع | Fork the repository
2. أنشئ فرع جديد | Create a new branch (`git checkout -b feature/amazing-feature`)
3. نفذ التغييرات | Commit your changes (`git commit -m 'Add amazing feature'`)
4. ارفع الفرع | Push to the branch (`git push origin feature/amazing-feature`)
5. افتح Pull Request | Open a Pull Request

## إرشادات | Guidelines

- تأكد من أن الكود يعمل بشكل صحيح | Ensure the code works correctly
- اكتب رسائل commit واضحة | Write clear commit messages
- احترم تصميم المشروع الحالي | Respect the existing project design

## الإبلاغ عن المشاكل | Reporting Issues

إذا وجدت خطأ أو لديك اقتراح، يرجى فتح Issue جديد مع وصف واضح.
If you find a bug or have a suggestion, please open a new Issue with a clear description.
