# Contributing Guidelines / إرشادات المساهمة

Thank you for considering contributing to this project! / شكرًا لاهتمامك بالمساهمة في هذا المشروع!

## How to Contribute / كيف تساهم

1. Fork the repository / اعمل Fork للريبو
2. Create your feature branch / أنشئ فرع للميزة الجديدة (`git checkout -b feature/amazing-feature`)
3. Commit your changes / احفظ التغييرات (`git commit -m 'Add amazing feature'`)
4. Push to the branch / ارفع التغييرات (`git push origin feature/amazing-feature`)
5. Open a Pull Request / افتح Pull Request

## Code of Conduct / مدونة السلوك

- Be respectful / كن محترمًا
- Provide constructive feedback / قدم ملاحظات بناءة
- Follow the project's coding standards / اتبع معايير البرمجة في المشروع
